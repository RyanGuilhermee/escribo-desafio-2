# Desafio Técnico Escribo 2

## Setup

### Requisitos

```
Flutter versão 3.16.1
```

### Instalando as dependências

```bash
$ flutter pub get
```

### Executando

**Modo debugging:**

```bash
$ flutter run lib/main.dart
```

**Release:**

```
Instale o apk que está na raiz do projeto (apk/app-release.apk)
```
